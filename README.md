# Projectos

- **Programação**: Scripts, programas e exercícios em várias linguagens.  
- **Web**: Sites, páginas interativas, APIs e bases de dados.  
- **Apps**: Protótipos, demos e aplicações completas.  
- **Documentação**: Guias e instruções para acompanhar os projetos.
