Informações dos Algoritmos

Cifra de César

Desloca cada letra por um número fixo de posições no alfabeto. Ex: A→D, B→E com deslocamento 3.

Cifra de Vigenère

Usa uma palavra-chave para deslocar letras de forma variável. Mais segura que César.

Base64

Codificação que converte dados binários em texto ASCII usando 64 caracteres.

Hexadecimal

Representa cada caractere como um valor hexadecimal de 2 dígitos.

Binário

Converte cada caractere em sua representação binária de 8 bits.

Código Morse

Sistema de codificação usando pontos e traços para representar letras e números.

Rail Fence

Escreve o texto em linhas alternadas e lê em colunas para criar a cifra.

Atbash

Substitui cada letra pela sua inversa no alfabeto: A↔Z, B↔Y, C↔X, etc.
